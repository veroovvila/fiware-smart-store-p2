# FIWARE Smart Store - P2

Práctica 2: Sistema inteligente de tienda con FIWARE.

## Descripción

Este proyecto implementa una solución para la Práctica 2 del curso de FIWARE.

## Requisitos

- Python 3.8+
- Docker & Docker Compose
- FIWARE components

## Instalación

```bash
git clone https://github.com/[usuario]/fiware-smart-store-p2.git
cd fiware-smart-store-p2
python -m venv .venv
source .venv/bin/activate
```

## Desarrollo

```bash
docker-compose up -d
```
